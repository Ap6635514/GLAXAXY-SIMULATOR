# 🌌 Galaxy Simulator

A Python-based N-body simulation that models galaxy formation using gravitational interactions.

## 🚀 Features
- Star-to-star gravity
- Central black hole
- Orbital motion
- Real-time simulation using Pygame

## 🧠 Concepts Used
- Newtonian Gravity
- N-body Simulation
- Orbital Mechanics

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
